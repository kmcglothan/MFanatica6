Jamroom5 jrAparna Module

Description
  • A simple module with just a tool to create new datastore modules with just a 'title' field.
  • The new module can then be customised with the Form Designer to add additional fields so as to turn it into something more specific, eg. a classified ads module.
  • Aparna is useful for site admins to create simple datastore items for their users without any knowledge of php and mysql.
Installation
  • Copy the module folder to your /modules folder.
  • Login as admin and this module should show under the modules Tools category.
  • It shouldn't be necessary to run the Integrity Check, but if there are any issues, do it anyway.
Usage
  • In the ACP click on Aparna's Tools tag, then the Clone button.
  • After a new module is created, run the Integrity Check and it should then appear in the ACP Tools section
  • Enable the new module and optionally edit its category so that it shows where needed in the ACP
  • Enable and configure the module for its required quotas.
