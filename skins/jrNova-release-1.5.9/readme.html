Thank you for purchasing and using the Jamroom 5 Nova skin.

If you are installing this template yourself please follow the instructions below using suitable file editing software such as BBEdit or alternatively use the Template Editor in your Jamroom Admin Tools.

If you prefer not to install yourself we have a guaranteed installation service available, visit us for more info: http://www.jamroom.net

Installation:

1. Unzip the file on your hard drive and using a suitable FTP software simply upload the jrNova folder to the skins directory on your server.

2. To activate the Nova Skin for your Jamroom5 Install, log-in as admin and click the Skins tab, then click the jrNova menu entry and on the Info tab check the set as active skin checkbox and finally click the save button.


Site branding:

Simply edit or replace the following files to change the branding of your site.

  • Main Header Logo (site wide | width: 325px height: 75px) - skins/jrNova/img/logo.png
  • Privacy Policy - skins/jrNova/privacy_policy.tpl
  • Terms Of Service - skins/jrNova/terms_of_service.tpl
  • Contact Us - The contact us page is a form, to modify, add/delete fields to the contact form, click the form designer button at the top right while on the contact us page logged in as the site admin.
  • Footer Information - skins/jrNova/footer.tpl

Nova Settings (skins/jrNova/config.php file):

Nova has many configuration settings, as admin you will see a link in the ACP drop down menu which links to the Nova Skin Settings page in your admin.

Note: Please take your time to view the Nova Settings and descriptions.


Artist & Member Quota ID's:

Set these two fields with your artist and member quota ID's.

Note: If Jamroom5 has just been installed, you'll want to create the member quota in your admin control panels Profiles > User Profiles > Tools > Quota Browser. After creating the new quota, if you did not use the quota clone tool, you'll want to modify all the modules you want to allow access for the new quota as they are not set by default.


Spotlight

Enter the 6 artist ID's you want to show in top spotlight section.

Note: Separate multiple ID's with a comma, ie. 1,2,3,4,5,6. If left blank the spotlight will show random artists.


Featured Tabs

You may choose which Artists and Media shows in the featured tabs on the index page.

  • Featured Artist ID's - Enter the artist ID's you want to show in featured artists tab section.
  • Featured Song ID's - Enter the song ID's you want to show in featured songs tab section.
  • Featured Video ID's - Enter the video ID's you want to show in featured videos tab section.

Note: Enter up to 3 ID's separated by a comma. ie. 1,2,3. If left blank the Featured tabs will show 3 random entries.


Site News ID

Enter the profile ID you want to use for the Site News section on the home page.

Note: If left blank, the Site News will use the default Admin ID which is 1. Also, when creating the blog for the site news, the blog category must be set to site news.


House Radio:

As standard the House Radio is DISABLED in the Nova Settings. If you haven't already done so, create a new Admin Audio Playlist by clicking the Add To Playlist button next to the audio tracks you want on your House Radio and make a note of the Playlist Name. After you've created your playlist, go to your Nova Settings, ENABLE the Show House Radio feature and enter the Playlist Name.

Note: You must have the jrPlaylist module installed and activated.


House Channel:

As standard the House Channel is DISABLED in the Nova Settings. If you haven't already done so, create a new Admin Video Playlist by clicking the Add To Playlist button next to the video tracks you want on your House Channel and make a note of the Playlist Name. After you've created your playlist, go to your Nova Settings, ENABLE the Show House Channel feature and enter the Playlist Name.

Note: You must have the jrPlaylist module installed and activated.


Auto Play

Enabling this option will turn on your players auto play feature.

Note: This is for the following profile players only. Audio, Playlist and Video.


Site Ads:

Enter your Ad code here for the top 468x60 Ad and the side 180x150 Ad. Or check the "Use Google Ads" box and enter your Google Adsense ID in the Google ID field. You may also hide all ads by checking the "Hide Ads" checkbox.

Note: As of now, the Jamroom 5 does NOT have an Ad Manager Module.


Require Images:

Enabling this option will hide entries without an image associated.


Page Breaks:

The following pagebreaks can be set.

  • Default Pagebreak - The default pagebreak for most lists.
  • Artist Pagebreak - This is the pagebreak for the Artist page.

Note: The pagebreaks for Artist should be divisible by 6!


Show Stats:

Enabling this option will show the stats at the bottom of the home page.

Note: Disable this option to hide the stats.
